인트로 화면(splash screen) 만들기

  1. 앱을 처음 실행할 때 로고나 이미지를 잠깐동안 표시하는 인트로 화면(splash screen)을 만들어보겠습니다.
  2. flutter_native_splash 같은 패키지를 이용할 수도 있지만 저는 안드로이드, iOS 네이티브에 직접 만드는 쪽을 택했습니다.

Adobe XD 에서 이미지 추출하기

책벗의 인트로 화면은 심플하게 하얀 배경에 로고 이미지를 띄우는 것으로 정했습니다. 아래 그림과 같습니다.




그러면 먼저 AdobeXD에서 이미지를 추출하겠습니다. 네이티브 방식을 따르기로 했으니 iOS용, 안드로이드용 이미지를 각각 추출합니다. 아래와 같이 로고이미지를 선택하고 레이어에서 이름을 splash_image로 변경했습니다.




파일 - 내보내기 - 선택됨을 누르거나 단축키인 Ctrl+E를 눌러줍니다.




에셋 내보내기 화면이 나오면 iOS, Android를 각각 선택해서 이미지를 내보냅니다. 해상도별로 iOS용으로는 1x, 2x, 3x 3종류의 이미지가 나오고 안드로이드는 ldpi부터 xxxhdpi까지 6종류의 이미지가 내보내집니다.

xml 파일 수정

먼저 mipmap과 동일한 위치인 res 디렉토리에 있는 AndroidManifest.xml 파일을 엽니다. 해당 파일에 아래와 같은 내용이 있다면 삭제해줍니다. flutter 2.5부터는 해당 내용이 deprecated 되었으므로 그대로 두면 “A splash screen was provided to Flutter, but this is deprecated” 라는 에러 메시지가 표시됩니다.

하단의 코드를 전부 삭제해주시면 됩니다.


다음엔 splash screen을 실제로 표시하게될 launch_background.xml을 수정합니다. 안드로이드 API 21 이상을 쓰면 drawable-v21 디렉토리의 launch_background.xml을 사용하고, 21미만은 drawable 디렉토리의 파일을 사용하므로 그냥 두개다 수정합니다.

저는 하얀 배경을 쓰니 배경색은 white를 그대로 두고, 가운데 들어갈 이미지만 수정해줬습니다. splash_image 뒤에 .png는 붙이지 않습니다.

여기까지 하면 안드로이드 인트로 화면표시는 완료되었습니다.
